# Medieval Overhaul - Advanced Mining

Integrates Mines 2.0 with Medieval Overhaul. Renames the "Minecraft" research to "Advanced Mining" and changes its prerequisite from Smithing to Medieval Overhaul's Mining research. The Mines 2.0 mining recipes are added to Medieval Overhaul's mine shaft, and the redundant Mines 2.0 "mine" building is hidden from the build menu and research tree. Requires both Mines 2.0 and Medieval Overhaul.
